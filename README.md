## 🚀 GitHub Pages konfiguratu

1. Fork edo klonatu errepositorioa
2. `_config.yml` fitxategian **zure GitHub erabiltzaile izena** eta **repositorio izena** aldatu:
   ```yaml
   baseurl: "/KAREAGA-2026"
   url: "https://axpirina.github.io"
   ```
3. GitHub → **Settings → Pages → Source: main branch / root** konfiguratu
4. Orri publikoa: `https://axpirina.github.io/KAREAGA-2026`
